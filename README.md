# ML_14lab

ОБЯЗАТЕЛЬНО К ПРОЧТНЕНИЮ ДЛЯ ОЗНОКОМЛЕНИЯ

Этот проект был создан для написания кода аналитической модели, которая способна определять ключевые параметры влияющие на стоимость автомобилей BMW.
Будут представлены 3 модели и в конце будут созданы 3 графика с их сравнением.

Код представляет собой полный рабочий процесс для подготовки данных, их анализа и построения моделей машинного обучения, направленных на прогнозирование цены автомобилей BMW. Сначала загружается исходный датасет и приводится в порядок: удаляются дубликаты, отсутствующие значения заполняются подходящими статистическими показателями, чтобы данные стали полноценными и пригодными для анализа. Далее категориальные признаки преобразуются в числовой формат с помощью one-hot кодирования - это необходимо, потому что большинство алгоритмов машинного обучения не работают с текстовыми категориями напрямую.

После предварительной подготовки данных проводится оценка мультиколлинеарности между признаками с помощью коэффициента VIF. Это позволяет понять, не содержат ли признаки избыточную информацию и не мешают ли они моделям, особенно линейной регрессии, корректно выявлять закономерности.

Затем подготовленный датасет разделяется на обучающую и тестовую выборки, после чего запускается обучение трёх моделей: линейной регрессии, случайного леса и XGBoost. Каждая модель анализирует зависимость между характеристиками автомобилей и их ценой, делает прогнозы и оценивается по стандартным метрикам качества - MAE, RMSE и коэффициенту детерминации R². Это позволяет сравнить модели между собой и определить, какая из них лучше справляется с задачей.

Кроме моделирования, код формирует два визуальных графика: зависимость цены от года выпуска и зависимость цены от пробега. Эти графики помогают увидеть основные тенденции в данных и визуально оценить, какие факторы сильнее всего влияют на стоимость автомобиля.

В конце все результаты сохраняются в отдельную папку: отчёт о мультиколлинеарности, метрики моделей, графики и преобразованный датасет. Таким образом, код автоматизирует весь цикл анализа - от очистки данных до обучения моделей и формирования наглядных итогов, позволяя получить полную картину того, как характеристики BMW влияют на их цену.
